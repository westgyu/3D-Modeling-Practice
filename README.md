# 3D 모델링 연습

## 갑자기 3D 모델링 하는 이유
1. 복잡한(?) 프로그래밍 세계에서 빠져나와 힐링을 찾기 위해서
2. 지필고사에서 벗어나 다른 활동을 해 보고 싶어서
3. 뜬금없이 갑자기 3D 모델링에 빠져서
4. 머리속 상상하는 것응 눈 앞에, 현실에서 직접 만져보고 사용하고 싶어서
